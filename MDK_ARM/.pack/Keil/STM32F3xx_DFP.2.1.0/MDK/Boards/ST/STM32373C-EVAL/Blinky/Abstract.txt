The 'Blinky' project is a simple demo program for
ST 'STM32F373VC' microcontroller using ST 'STM32373C-EVAL' Evaluation Board.
Compliant to Cortex Microcontroller Software Interface Standard (CMSIS).

Example functionality:
 - Clock Settings:
   - XTAL    =            8.00 MHz
   - SYSCLK  =           72.00 MHz

 - LEDs blink with speed depending on potentiometer.
 - blinking is paused while holding down Wakeup button.


The Blinky program is available in different targets:
 - Debug:
   - Compiler optimization Level 1
   - Enabled Event Recorder
   - Keil RTX5 variant 'Source'
 - Release:
   - Compiler optimization Level 3
   - Keil RTX5 variant 'Library'
